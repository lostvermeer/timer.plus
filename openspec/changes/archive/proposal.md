# Add TimerPlus MVP

## Summary
TimerPlus MVP is a web interval timer app where users can create a routine with work time, rest time, and rounds, then start, pause, resume, reset, and save that routine locally.

## Why
I want to build my own Timer Plus-style app for workouts, focus sessions, and repeated routines.

## What Changes
- Add a timer routine builder.
- Add a timer runner screen.
- Support work time, rest time, and rounds.
- Support start, pause, resume, and reset.
- Save routines locally.
